Bienvenido a: Aprende Fusion para CAD en 90 Minutos

Transcripción y Objetivos de Aprendizaje

En este curso, aprenderás las habilidades básicas para usar Fusion en el diseño asistido por computadora. En aproximadamente 90 minutos, aprenderás a navegar por la interfaz de Fusion y a crear y gestionar archivos de diseño.

Descubrirás cómo crear bocetos totalmente restringidos, crear componentes y ensamblajes, modificar modelos 3D, generar renders de productos y exportar archivos de diseño para impresión 3D.

Usa los conjuntos de datos adjuntos para seguirnos mientras creamos un vaso para bebidas. Para práctica adicional, utiliza los ejercicios y conjuntos de datos para crear un posavasos, una mesa y un render con tus diseños.

Al completar este curso, serás capaz de:

  • Navegar por la interfaz de Fusion.
  • Usar las herramientas de navegación y visualización.
  • Crear y gestionar archivos de diseño.
  • Crear bocetos y crear modelos 3D a partir de bocetos.
  • Modificar piezas con características clave, incluyendo extrusión, revolución, vaciado y barrido.
  • Crear componentes y aplicar uniones de ensamblaje para un movimiento adecuado.
  • Crear un dibujo dimensionado de un componente.
  • Aplicar configuraciones visuales y renderizar un entorno de proyecto.
  • Compartir diseños y exportar archivos de diseño para impresión 3D.
